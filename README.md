# SQLite CRUD Management System

A Python-based student management system implementing complete CRUD operations using SQLite.

---

## Features

- Create records
- Read/search records
- Update existing data
- Delete records
- Parameterized queries
- Input validation

---

## Tech Stack

- Python
- SQLite
- Python DB-API
- OOP Concepts

---

## Database Concepts Used

- Schema normalization
- Primary keys
- Query optimization
- Safe query execution

---

## Example Operations

### Add Student

```sql
INSERT INTO students(name, age)
VALUES (?, ?)
```

### Fetch Records

```sql
SELECT * FROM students
```
